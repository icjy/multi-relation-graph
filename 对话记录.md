## 1. 项目目标

用户希望实现一个本地可用的反欺诈关系图谱分析工具：

- 支持前端上传 CSV / Excel 文件
- 基于贷款明细构建用户、中介、设备、IP、地址簇等多关系图谱
- 计算二度风险指标
- 识别疑似中介团伙
- 生成社区特征、中心性特征和策略特征宽表
- 宽表可导出 CSV / Excel
- 支持服务版页面和可离线打开的 `snapshot.html`

## 2. 输入字段口径

当前输入字段主要包括：

```text
loan_task_id
apply_time
final_result
reloan_flag
return_flag
app_key
app_user_id
consigneeAddr
consigneeMobileId
ip
device_id
receiverAddr
addr_cluster_id
fpd/cpd/spd/tpd 系列逾期字段
```

字段含义确认：

- `app_user_id`：借款人 / 用户
- `consigneeMobileId`：收货人手机号 / 中介
- `consigneeAddr`：原始地址
- `receiverAddr`：预处理后的地址
- `addr_cluster_id`：地址聚类 ID
- `device_id`：设备号
- `ip`：申请 IP
- `reloan_flag`：首复贷标识
- `return_flag`：是否退货，`1` 表示已退货
- `final_result=30`：借款通过 / 已放款样本

坏账率和贷款笔数类特征只统计 `final_result=30` 的通过样本；申请笔数类特征统计筛选后的全量行。

## 3. 二度指标口径

### 二度中介数

对某个中介：

```text
先找到它的一度借款人
再通过这些借款人反查其他中介
去掉自己
统计去重后的中介数
```

### 二度借款人数

对某个借款人：

```text
先找到它的一度中介
再通过这些中介反查其他借款人
去掉自己
统计去重后的借款人数
```

### 二度坏账率 / 二度逾期率

对某个借款人：

```text
找到二度借款人集合
取这些借款人的贷款记录
只保留 final_result=30
用逾期笔数 / 通过贷款总笔数计算
```

## 4. 社区算法讨论与实现

最初实现了基于共享用户的中介投影图连通分量。

后来确认需要保留该方式，并新增：

- Louvain
- Leiden

### 连通分量

当前连通分量定义为：

```text
共享用户中介投影图上的连通分量
```

即：

```text
如果两个中介共享同一个 app_user_id，则两个中介连边。
只要一组中介通过共享用户链路可以互相连通，就归为同一个团伙。
```

### Louvain / Leiden

Louvain 和 Leiden 的逻辑是：

```text
User - Agent
User - Device
User - IP
User - AddressCluster
```

先投影成用户同构图，再运行社区发现算法。

当前 Leiden 是离线可用的 Leiden-like 实现：

```text
先运行 Louvain
再把社区内部不连通的部分拆开
最后重新编号社区
```

原因是要保证 `snapshot.html` 单文件离线可用，不依赖外部包。

## 6. 中介中心性解释

用户询问 `中介中心性(共享用户)` 是否表示“两个借款人通过一个中介关联”。

确认：

当前字段不是二部图中的用户桥接数，而是中介投影图上的介数中心性。

也就是：

```text
衡量某个中介是否位于其他中介之间的最短路径上。
```

例如：

```text
A -- B -- C
```

中间的 `B` 可能有较高中介中心性。

如果只有：

```text
A -- B
```

两个节点直接相连，没有第三个节点需要通过 A 或 B，因此介数中心性为 0。

## 7. 社区风险分口径

社区风险分是规则加权分，不是模型概率。

### 连通分量 / 中介社区

```text
社区风险分 =
坏账率 * 55
+ 网络密度 * 20
+ 共享借款人占比 * 15
+ 团伙规模因子 * 10
```

其中：

```text
团伙规模因子 = min(中介数 / 10, 1)
```

### Louvain / Leiden / 用户社区

```text
社区风险分 =
坏账率 * 55
+ 网络密度 * 15
+ 设备密度 * 10
+ IP密度 * 10
+ 地址密度 * 10
```

## 8. 图谱交互优化

用户提出大团伙节点很多，看不清边。

讨论过的方案包括：

- 点击节点高亮邻居
- 节点搜索聚焦
- 节点类型开关
- Top N 关键节点展示
- 社区折叠
- 邻接明细表
- 路径查询
- 力导向图 / 缩放拖拽

最终实现：

- 单击节点：高亮该节点及一度邻居，其他节点和边变淡
- 双击节点：以该节点为中心重新查询图谱
- 图例变成节点类型开关，可隐藏 / 显示中介、借款人、设备、IP、地址簇
- 鼠标滚轮缩放图谱
- 拖拽图谱任意位置进行平移
- 边按关系类型上色
